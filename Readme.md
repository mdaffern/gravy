
# gravy

```js
gravy('project-name', 'user-name', 'access-key')
  .browser('ie9..11')
  .browser('safari')
  .browser('chrome')
  .browser('firefox')
  .reporter(gravy.reporters.grid())
  .test('http://localhost:3000');
```

## Installation

  Install with [component(1)](http://component.io):

    $ component install yields/gravy
    $ npm install gravy
